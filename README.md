### Dependencies
- Python 3.7
- Pipenv
- Jupyter
- iPython

### Getting Started
1. Clone this repository
2. Install the dependencies listed above
3. Run `pipenv install`
4. Enter the virtual environment with `pipenv shell`
5. Check the name of your virtual environment in paretheses (ENV-NAME). Run `python -m ipykernel install --user --name=ENV-NAME`
6. Run `jupyter notebook`
7. When opening a file, ensure that you are using the kernel for your new virtual environment (ENV-NAME)
